Edit

add

Add a file to the git index (aka staging area)

paramtype [= default]description
corestring = 'default'The plugin core identifier to use for plugin injection
fs [deprecated]FileSystemThe filesystem containing the git repo. Overrides the fs provided by the plugin system.
dirstringThe working tree directory path
gitdirstring = join(dir, '.git')The git directory path
filepathstringThe path to the file to add to the index
returnPromise<void>Resolves successfully once the git index has been updated

Example Code:

await new Promise((resolve, reject) => fs.writeFile(
  '$input((/README.md))',
  `$textarea((# TEST))`,
  (err) => err ? reject(err) : resolve()
))
await git.add({ dir: '$input((/))', filepath: '$input((README.md))' })
console.log('done')
